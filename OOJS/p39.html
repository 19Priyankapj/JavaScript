javascript encapsulation

it is process of binding the data with the function acting on that data. allows us to control the data and validate it

points to remember...
  • use let and const keyword to make data members private
  • use setter method to set the data and getter method to get the data

encapsulation allows us to handle an object using the following properties

read/write - here, we use setter method to write the data and getter method is used to read the data

read only- in this case we use getter method only

write only-in this case we use setter method only
