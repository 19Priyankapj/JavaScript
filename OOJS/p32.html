static method

js provides ststic methods that belongs to the class instead of an instance of an that class these methods are called directly on the class itself

points to remember...

  • the ststic keyword is used to declare a static method
  • the ststic method can be any name
  • a class can contain more than one static method
  • if we declare more than one ststic method with similar name, the js always invokes the last one
  • we can use this keyword to call a static method within the another static method
  • we cant use this keyword directly to call ststic method using within non-static method
  • in such case we can call this ststic method either using the class name or the property of the constructor
